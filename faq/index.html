To View All Linked Images In a Page

  • Right Click on web page , and select Build Gallery From All Image Links
  • All the linked images from the web page will be opened in a new page

To View All High Res Images In a Page

  • Right Click on web page , and select Build Gallery From All High Res Images in page
  • All the High Resolution images from the web page will be opened in a new page
